1.5 Block和迭代器

Block

block_name{
   statement1
   statement2
   ..........
}

def test
   puts "在 test 方法内"
   yield
   puts "你又回到了 test 方法内"
   yield
end
test {puts "你在块内"}

Block可以接收来自yield的参数

def test
   yield 5
   puts "在 test 方法内"
   yield 100
end
test {|i| puts "你在块 #{i} 内"}

迭代器

each 迭代器

ary = [1,2,3,4,5]
ary.each do |i|
   puts i
end

collect 迭代器 

a = [1,2,3,4,5]
b = Array.new
b = a.collect{ |x|x }
puts b

find 迭代器

原理大致是

class Array
    def find
        for i in 0...size
            return self[i] if yield self[i]
        end
    end
end

a = [1,2,3,4,5]
b = a.find{ |x| x==3 }
puts b
