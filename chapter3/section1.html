3.1 字符串

字符串中嵌入变量

双引号字符串可以内插变量,里面支持常用的一些操作

name1 = "Joe"
name2 = "Mary"
puts "你好 #{name1},  #{name2} 在哪?"
puts "#{'EPAM ' * 3}"

使用%q 和 Q%创建变量

Ruby 中还支持一种采用 %q 和 %Q 来引导的字符串变量，%q 使用的是单引号引用规则，而 %Q 是双引号引用规则，后面再接一个 (! [ { 等等的开始界定符和与 } ] ) 等等的末尾界定符。

desc1 = %Q{Ruby 的字符串可以使用 '' 和 ""。}
desc2 = %q|Ruby 的字符串可以使用 '' 和 ""。|

puts desc1
puts desc2

字符串常用操作

  1. str * integer 重复字符串
  2. str + other_str 连接字符串
  3. str.capitalize 把字符串转换为大写字母显示。
  4. str.capitalize! 把字符串转换为大写字母显示
  5. str.to_f 转浮点数
  6. str.to_i 转整数
  7. str.upto(other_str) { |s| block } 遍历字符串中的字符
  8. str.upcase 转大写
  9. str.upcase! 转大写
  10. str.downcase 转小写
  11. str.downcase 转小写
  12. str.length 字符串长度
  13. str.replace 字符串替换
  14. str.reverse 字符串倒序
  15. str.reverse! 字符串倒序
  16. str.empty? 字符串判空
  17. str.eql? 字符串相等
