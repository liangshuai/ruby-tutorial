Ruby简介

Ruby是一种纯粹的面向对象编程语言。它由日本的松本行弘（まつもとゆきひろ/Yukihiro Matsumoto）创建于1993年。 您可以在 www.ruby-lang.org 的 Ruby 邮件列表上找到松本行弘（まつもとゆきひろ/Yukihiro Matsumoto）的名字。在 Ruby 社区，松本也被称为马茨（Matz）。 Ruby 是"程序员的最佳朋友"。 Ruby 的特性与 Smalltalk、Perl 和 Python 类似。Perl、Python 和 Smalltalk 是脚本语言。Smalltalk 是一个真正的面向对象语言。Ruby，与 Smalltalk 一样，是一个完美的面向对象语言。使用 Ruby 的语法比使用 Smalltalk 的语法要容易得多。

Ruby特性

  1. Ruby 是开源的
  2. Ruby 是一种通用的、解释的编程语言。
  3. Ruby 是一种真正的面向对象编程语言。
  4. Ruby 是一种类似于 Python 和 Perl 的服务器端脚本语言。
  5. Ruby 拥有Rails、Sinatra等优秀的Web开发框架，还有众多优秀的测试框架等
  6. Ruby 可以用来编写通用网关接口（CGI）脚本。
  7. Ruby 可以被嵌入到超文本标记语言（HTML）。
  8. Ruby 语法简单，这使得新的开发人员能够快速轻松地学习 Ruby。
  9. Ruby 与 C++ 和 Perl 等许多编程语言有着类似的语法。
  10. Ruby 可扩展性强，用 Ruby 编写的大程序易于维护。
  11. Ruby 可用于开发的 Internet 和 Intranet 应用程序。
  12. Ruby 有丰富的内置函数，可以直接在 Ruby 脚本中使用。
  13. Ruby DSL表达力好
